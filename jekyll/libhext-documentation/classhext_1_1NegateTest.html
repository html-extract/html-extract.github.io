Detailed Description

Negates the result of another ValueTest.

Example:
NegateTest not_foo(std::make_unique<EqualsTest>("foo"));
assert( not_foo.test("bar"));
assert( not_foo.test("baz"));
assert(!not_foo.test("foo"));
